# MileIQ Clone – PWA

A full-featured mileage tracker PWA. Works offline, installs to iPhone home screen.

## Deploy to Vercel (Free – 3 steps)

1. Go to https://vercel.com and sign up free with GitHub
2. Click "Add New Project" → Import this folder
3. Click Deploy — done! You get a live HTTPS URL.

## Install on iPhone

1. Open your Vercel URL in **Safari** (must be Safari, not Chrome)
2. Tap the **Share** button (box with arrow at bottom of screen)
3. Scroll down → tap **"Add to Home Screen"**
4. Tap **Add** — the MileIQ icon appears on your home screen!

## Features
- GPS real-time drive tracking
- Swipe right = Business, left = Personal
- Auto-classify by work hours
- Vehicle fuel efficiency settings
- Live fuel price calculator
- CRA-compliant mileage reports
- Export to CSV
- Offline support via Service Worker
- All data saved locally (no account needed)

## Local Development
```
npm install
npm start
```
